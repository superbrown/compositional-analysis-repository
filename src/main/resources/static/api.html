data-repository-app REST endpoints

Cateogy Data

api/v01/dataCategory/{dataCategoryId} [GET] [returns JSON]
api/v01/dataCategory?dataCategoryName={dataCateogryName} [GET] [returns JSON]
api/v01/dataCategory/searchableColumnNames?dataCategoryName={dataCateogryName} [GET] [returns JSON]
api/v01/dataCategory/names/all [GET] [returns JSON]
api/v01/dataCategories/all [GET] [returns JSON]

Datasets

api/v01/addDataset/{dataset_id} {JSON in the request body} [POST]
api/v01/datasets/all [GET] [returns JSON]
api/v01/dataset/{datasetId}/sourceDocument [GET] [returns binary document]
api/v01/dataset/{datasetId}/attachments [GET] [returns binary document, a zip file]
api/v01/dataset/{datasetId}/rows {JSON in the request body} [GET]

Dataset Rows

api/v01/rows {query JSON in the request body} [POST] [returns JSON]
api/v01/rows/flat {query JSON in the request body} [POST] [returns JSON]
api/v01/rows/asFile {query JSON in the request body} [POST] [returns binary document]
api/v01/row/{rowId} [GET] [returns JSON]

Utilities

api/v01/getNamesOfSheetsWithinExcelWorkbook (multipart file in the request body) [POST] [returns JSON]

Admin

api/v01/addDataCategory?name={dataCateogryName} [GET] [returns JSON]
api/v01/removeDataset/{datasetId} [GET]
api/v01/dropDatabaseAndReIngestAllDataFromOriginallyUploadedFiles [GET] [returns JSON with a list of errors]
api/v01/attemptToCleanupDataFromAllPreviouslyIncompleteDatasetUploads [GET] [returns JSON with a list of errors]
