data-repository-app REST endpoints

Cateogy Data

api/dataCategory/{dataCategoryId} [GET] [returns JSON]
api/dataCategory?dataCategoryName={dataCateogryName} [GET] [returns JSON]
api/dataCategory/searchableColumnNames?dataCategoryName={dataCateogryName} [GET] [returns JSON]
api/dataCategory/names/all [GET] [returns JSON]
api/dataCategories/all [GET] [returns JSON]

Datasets

api/addDataset/{dataset_id} {JSON in the request body} [POST]
api/datasets/all [GET] [returns JSON]
api/dataset/{datasetId}/sourceDocument [GET] [returns binary document]
api/dataset/{datasetId}/attachments [GET] [returns binary document, a zip file]
api/dataset/{datasetId}/rows {JSON in the request body} [GET]

Dataset Rows

api/rows {query JSON in the request body} [POST] [returns JSON]
api/rows/flat {query JSON in the request body} [POST] [returns JSON]
api/rows/asFile {query JSON in the request body} [POST] [returns binary document]
api/row/{rowId} [GET] [returns JSON]

Utilities

api/getNamesOfSheetsWithinExcelWorkbook (multipart file in the request body) [POST] [returns JSON]

Admin

api/addDataCategory?name={dataCateogryName} [GET] [returns JSON]
api/removeDataset/{datasetId} [GET]
api/dropDatabaseAndReIngestAllDataFromOriginallyUploadedFiles [GET] [returns JSON with a list of errors]
api/attemptToCleanupDataFromAllPreviouslyIncompleteDatasetUploads [GET] [returns JSON with a list of errors]
